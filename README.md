# Simple LibVLC based music player

## Requirements
 * [Python](https://www.python.org) 3.5.x or higher
 * [LibVLC](https://wiki.videolan.org/LibVLC)

## Used Python packages
 * [python-vlc](https://pypi.org/project/python-vlc)
